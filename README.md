# BusDepotManagementSystem
 A SQL project demonstrating database design, queries, triggers, and stored procedures for a Bus Depot Management System.

 Bus Depot Management System 

 Overview
This project is a SQL-based Bus Depot Management System built for practice in MySQL Workbench.  
It demonstrates database design, joins, subqueries, triggers, and stored procedures.

 Features
- Bus, Driver, Route, Schedule, Ticket tables
- Sample data insertion
- Queries for revenue, busiest route, passenger details
- Advanced SQL: Views, Stored Procedures, Triggers

 Files
- schema.sql → Database schema
- data.sql → Sample data
- queries.sql → Practice queries
- advanced.sql → Advanced SQL features

 How to Run
1. Create database using `schema.sql`
2. Insert sample data using `data.sql`
3. Run queries from `queries.sql`
4. Test advanced features using `advanced.sql`
